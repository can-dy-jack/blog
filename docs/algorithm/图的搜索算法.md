并查集是用来检查两个顶点之间的连通性。而找出图所有的顶点呢、找出它两个顶点之间的路径等都需要深度优先搜索算法或广度优先搜索算法。

## 深度优先搜索算法

|                          |    时间复杂度    |  空间复杂度  |
| :----------------------: | :--------------: | :----------: |
|     DFS 遍历所有节点     |     $O(V+E)$     |    $O(V)$    |
| DFS 遍历两点之间所有路径 | $O((2^V)∗(V+E))$ | $O((2^V)∗V)$ |

`V` 表示顶点数，`E` 表示边数。
